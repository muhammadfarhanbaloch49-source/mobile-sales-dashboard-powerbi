# 📊 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Mobile Sales Dashboard** built in **Microsoft Power BI**. The dashboard provides valuable insights into mobile sales performance, revenue, customer purchasing behavior, brand performance, and regional sales trends.

The dashboard enables users to monitor key business metrics through interactive visualizations and filters, supporting data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze overall mobile sales performance.
- Monitor revenue and sales trends.
- Compare brand and model performance.
- Analyze customer purchasing behavior.
- Evaluate regional sales performance.
- Create an interactive Power BI dashboard for business reporting.

---

## 📂 Dataset Overview

The dataset includes information such as:

- Mobile Brand
- Mobile Model
- Sales Amount
- Quantity Sold
- Customer Details
- Payment Method
- City
- State
- Date of Sale
